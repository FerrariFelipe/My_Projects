Gerenciador de Produtos (CRUD em Python)

Este projeto consiste em uma aplicação de linha de comando (CLI) para gerenciar um catálogo de produtos. O sistema permite realizar as quatro operações básicas de manipulação de dados (CRUD: Create, Read, Update, Delete), utilizando estruturas fundamentais da linguagem Python.
